1. Introducción¶

1.1. Instalación¶

Para instalar GBNP, primero descarguemos de aquí el código fuente, luego descomprimos el archivo GBNP-1.0.3.tar.gz en el subdirectorio pkg de su instalación de GAP ( o en el subdirectorio pkg de cualquier otro directorio raíz de GAP, por ejemplo uno agregado con el argumento -l) con el siguiente comando: tar -xvzf GBNP-1.0.3.tar.gz.

Luego, GBNP se carga con el comando GAP

gap> LoadPackage( "GBNP" );

Aquellos que quieran descargar esta documentación pueden encontrarla en este link y extraerla con tar -xvzf GBNPdoc-1.0.3.tar.gz. También está incluido en el paquete.

1.2. Usos del paquete¶

Si deseamos calcular una base de Gröbner, creamos una lista de NP (polinomios no conmutativos en formato NP), como se describe en la Sección Polinomios no conmutativos (NPs). Esto se puede hacer directamente o mediante el uso de las funciones de transición descritas en la Sección Conversión de polinomios a diferentes formatos. Para ejecutar el algoritmo estándar, use las funciones de la Sección Funciones Gröbner, variante estándar. Con estas funciones, podemos intentar encontrar una base de Gröbner. La palabra try se incluye porque no se garantiza que el algoritmo para calcular las bases de Gröbner termine. Los problemas de impresión de polinomios en formato NP se analizan en la Sección Imprimiendo Polinomios en formato NP. Si se encuentra la base de Gröbner y la dimensión del álgebra del cociente \(Q\) (consulte la Sección Álgebras Cocientes) es finita, puede encontrar una base de monomios para \(Q\) con las funciones de la Sección Álgebras de cocientes de dimensión finita. Para un análisis más avanzado de \(Q\), como una prueba de dimensionalidad finita o infinita, o para determinar su crecimiento o su serie parcial de Hilbert, use las funciones de la Sección Finitud y series de Hilbert.

Hay tres variantes del algoritmo básico de Gröbner:

  • la versión truncada,

  • la versión de seguimiento y

  • la versión del módulo.

En el caso homogéneo (ponderado) (descrito en la Sección Variante de truncamiento), la versión truncada, dada por las funciones descritas en la Sección Funciones de la variante truncada, calcula la parte de una base de Gröbner hasta un peso indicado.

La versión de traza (descrita en la Sección Variante de rastreo), dada por las funciones descritas en la Sección Funciones de la variante de seguimiento, calcula una expresión de los polinomios de la base de Gröbner encontrados en términos de los generadores originales.

La versión del módulo (descrita en las Secciones Polynomios No Conmutativos para Modulos (NPMs), Variante de módulo y Registros de bases de Gröbner), dada por las funciones descritas en la Sección funciones-de-la-variante-del-módulo, calcula una base de Gröbner para un submódulo de un módulo \(Q\) libre de rango finito.

Pemos ver los archivos de ejemplo en el Capítulo A para inspirarnos. El código fuente de los archivos se puede examinar para funciones auxiliares, que a menudo se utilizan en las funciones principales, pero no se consideran necesarias para un usuario nuevo.
